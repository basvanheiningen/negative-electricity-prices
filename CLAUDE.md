# Architecture
- `/utils/entsoe` - Utility functions used to gather ENTSOE data
- `/data/entsoe` - Save data from API calls, also save a watermark so you know how far data is up-to-date
- `/data/entsoe/day_ahead_prices_NL` - Save data on day ahead prices (EPEX) for NL
- `/data/entsoe/energy_generation_NL` - Save data on energy generation for NL
